### 기술통계
 사용자가 선택한 분석 대상 항목 조건별로 평균, 중위수, 표준편차, 분산, 왜도, 첨도, 빈도 분석 결과를 제공합니다.<br> 
 여러 문항에 대한 기술통계를 동시에 도출하거나, 여러 문항을 차원 축소하여 분석할 수 있습니다.<br> 
 특정 척도나 기준에 맞는 데이터를 추출하여 개별 문항 또는 차원 축소된 상태의 문항에 대한 분석을 수행할 수 있습니다.<br> 
 예: 특정 조건의 응답자 데이터에 대해 항목별 또는 항목 간 기술통계 분석 수행합니다</strong>와 더불어 <strong>조사/분석 활용법</strong>을 통해 더욱 효과적으로 활용해 보세요!

#### ✔ 변수 설정방법  
  ➊ 분석대상 설정에서 분석할 데이터 변수(열)을 선택합니다.  
  ➋ 분석결과 항목에서 분석방법을 선택합니다.   
  ➌ 데이터 분석 아이콘을 클릭하여 결과를 확인합니다.  
    
#### ✔ 조건별 세부 설정
 성별이 남성이면서 직급이 상사인 경우"를 설정하는 방법을 예제로 설명하겠습니다.   
   
   ➊ 분석대상 설정에서 분석할 데이터 변수(열)을 선택합니다.   
   ➋ 조건별 세부설정을 클릭합니다.  
   ➌ 추출 범위 설정에서 성별에 해당하는 변수(열)과 해당 척도인 남성을 선택하고 추출범위에 추가합니다.   
   ➍ 추출조건에서 교집합 선택합니다.  
   ➎ 직급에 해당하는 변수(열)과 해당 척도인 상사를 선택하고 추출범위에 다시 추가합니다.  
   ⑥ 분석결과 항목에서 분석방법을 선택합니다.  
   ⑦ 데이터 분석 아이콘을 클릭하여 결과를 확인합니다.
