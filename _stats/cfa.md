**확인적 요인분석(CFA)**은 **이론적 모델**을 바탕으로 변수들 간의 관계를 검증하는 **통계적 기법**입니다. 이는 **탐색적 요인분석(Exploratory Factor Analysis, EFA)**과는 달리, 이미 특정 요인 구조가 존재한다고 가정하고, 그 구조가 데이터에 적합한지 검증하는 데 사용됩니다.

### 주요 목적
1. **이론적 모델 검증**: 기존에 제시된 **이론적 모델**을 데이터에 적용하여, 그 모델이 실제 데이터에 잘 맞는지를 평가합니다.
2. **구조적 관계 평가**: 변수들 간의 **잠재적 관계**(예: 특정 변수들이 특정 요인에 얼마나 기여하는지)를 검증합니다.
3. **모델 적합도 평가**: 모델의 **적합도**(model fit)를 평가하여, 제시된 요인 구조가 데이터에 잘 맞는지 판단합니다.

### 주요 요소
1. **잠재 변수(Latent Variables)**: 직접적으로 측정할 수 없는 변수들로, 보통 요인들이 잠재 변수로 간주됩니다. 예를 들어, **"만족도"**와 같은 추상적인 개념은 잠재 변수입니다.
2. **측정 변수(Observed Variables)**: 잠재 변수에 의해 영향을 받는 **측정 가능한 변수들**입니다. 예를 들어, "고객 만족도"는 여러 설문 항목을 통해 측정된 여러 변수들로 구성될 수 있습니다.
3. **모델 경로(Model Paths)**: 잠재 변수와 측정 변수 간의 관계를 나타내는 경로입니다. CFA는 이 경로가 실제 데이터에 어떻게 맞는지 검토합니다.

### 주요 과정
1. **모델 설정**: 연구자는 **이론적 가설**에 따라 어떤 변수들이 어떤 요인에 속하는지를 설정합니다. 예를 들어, "고객 만족도"라는 잠재 변수는 설문 항목들로 측정될 수 있습니다.
2. **모델 적합도 검사**: CFA는 설정된 모델이 실제 데이터에 적합한지를 확인합니다. 여러 **적합도 지표**(예: χ² 통계량, CFI, RMSEA 등)를 사용하여 모델의 적합도를 평가합니다.
3. **결과 해석**: 모델의 **경로 계수**(factor loadings)와 **적합도 지표**를 통해 이론적 모델이 데이터에 잘 맞는지 확인하고, 필요한 경우 모델을 수정합니다.

### 적합도 지표 (Model Fit Indices)
CFA에서 중요한 적합도 지표는 모델이 데이터에 얼마나 잘 맞는지를 평가하는 데 사용됩니다. 대표적인 지표는 다음과 같습니다:
- **Chi-square (χ²)**: 모델의 적합도를 평가하는 전통적인 지표. 값이 작을수록 좋지만, 샘플 크기와 상관이 있습니다.
- **CFI (Comparative Fit Index)**: 0과 1 사이의 값으로, 0.90 이상이 일반적으로 적합한 모델을 의미합니다.
- **RMSEA (Root Mean Square Error of Approximation)**: 0.08 이하가 적합한 모델을 나타냅니다.
- **TLI (Tucker-Lewis Index)**: 0.90 이상이 적합도를 나타냅니다.

### 장점
1. **이론 검증**: 기존 이론이나 가설을 데이터로 검증할 수 있습니다.
2. **모델 수정 가능**: 모델의 적합도가 낮을 경우, 경로를 수정하거나 잠재 변수를 추가하여 개선할 수 있습니다.
3. **측정 모델과 구조 모델 분리**: CFA는 **측정 모델**(변수들이 어떤 요인에 속하는지)과 **구조 모델**(요인 간의 관계)을 분리하여 분석합니다.

### 활용 예시
- **심리학**: 특정 심리적 특성(예: 자기효능감, 스트레스 등)을 측정하기 위한 잠재 변수와 그 측정 항목들 간의 관계를 검증합니다.
- **마케팅**: 소비자의 브랜드 충성도나 만족도를 측정하기 위해 다양한 설문 항목과 그 요인 간의 관계를 평가합니다.
- **사회과학**: 특정 사회적 개념을 측정하는 여러 변수들을 기반으로, 해당 개념이 실제로 존재하는지를 확인합니다.

##### Parameter Estimates
다음은 각 변수와 요인 간의 관계를 나타내며, 살펴봐야 할 주요 사항은 다음과 같습니다.

1. **Factor Loadings (요인 부하)**:
   - **est (추정값)**이 크고, **p-값**이 0.05 이하일 경우, 해당 변수와 잠재변수 간의 관계가 유의미하다는 것을 나타냅니다.
   - **std.lv (표준화된 잠재변수 추정값)**을 확인하여 잠재변수와 관측변수 간의 관계 강도를 비교합니다.

2. **Standard Error (표준 오차)**:
   - **se**가 작을수록 추정값이 정확하다는 의미입니다.

3. **Z-value (Z-값)**:
   - **z-값**이 크면 해당 추정값이 통계적으로 유의미하다는 것을 의미합니다.

4. **p-값**:
   - **p-값**이 0.05 이하이면 해당 파라미터가 **통계적으로 유의미**하다는 뜻입니다.

5. **Standardized All (모든 변수의 표준화 값)**:
   - **std.all**을 통해 모든 변수의 상대적인 영향을 비교할 수 있습니다.

![Image](https://github.com/user-attachments/assets/6df241b0-aa68-4c0d-9a77-73bf5d9dea9e)  
| **변수**       | **설명**                                                                 |
|----------------|--------------------------------------------------------------------------|
| **lhs (Left-hand side)** | **좌측 변수**: 모델에서 설명하려는 대상 변수입니다. 즉, 관측변수 또는 종속변수입니다. |
| **op (Operator)** | **연산자**: 모델에서 좌측과 우측 변수를 어떻게 연결할지 나타내는 연산자입니다. 예를 들어, `=`, `~`, `+` 등이 있습니다. |
| **rhs (Right-hand side)** | **우측 변수**: 모델에서 독립변수나 잠재변수를 나타내며, 좌측 변수에 영향을 미치는 요인입니다. |
| **exo (Exogenous variables)** | **외생 변수**: 모델에서 다른 변수에 영향을 주지만 영향을 받지 않는 변수입니다. 일반적으로 독립변수나 잠재변수로 사용됩니다. |
| **est (Estimate)** | **추정값**: 모델을 통해 계산된 파라미터 값입니다. 예를 들어, 요인 부하값, 회귀 계수 등이 이에 해당합니다. |
| **se (Standard Error)** | **표준 오차**: 추정값의 정확도를 나타내며, 값이 작을수록 추정값이 정확하다는 의미입니다. |
| **z (Z-value)** | **Z값**: 추정값을 표준 오차로 나눈 값으로, 추정값이 얼마나 유의미한지를 평가하는 데 사용됩니다. |
| **pvalue** | **p-값**: 추정값이 통계적으로 유의미한지를 나타냅니다. 일반적으로 0.05 이하이면 유의미한 것으로 판단합니다. |
| **std.lv (Standardized Latent Variable)** | **표준화된 잠재변수 추정값**: 잠재변수에 대한 **표준화된 추정값**으로, 각 잠재변수가 해당 요인에 얼마나 부합하는지를 나타냅니다. |
| **std.all (Standardized All)** | **모든 변수에 대한 표준화된 값**: 모델 내 모든 변수의 표준화된 값을 나타내며, 각 변수의 상대적인 영향력을 비교할 수 있게 해줍니다. |

##### 경로모델 시각화
확인적 요인 분석모델을 경로 다이어그램 형태로 시각화로 나타낸 것으로서, 잠재변수(요인)와 관측변수 간의 관계를 화살표로 나타낸것 입니다.
