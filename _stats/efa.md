## 개요

탐색적 요인분석(Exploratory Factor Analysis, EFA)은 변수들 간의 상관관계를 분석하여 숨겨진 요인(latent factors) 을 탐색하는 통계 기법입니다.   
즉, 다수의 관측 변수를 몇 개의 공통된 요인으로 축소하여 데이터 구조를 이해하는 데 사용됩니다.

### ❓ 탐색적 요인이란?  
탐색적 요인(Exploratory Factor)은 **탐색적 요인 분석(EFA)**에서 사용되는 용어로, 데이터 내에 존재할 수 있는 잠재적인 요인들을 찾아내는 과정입니다. EFA는 데이터를 통해 특정한 요인이나 차원을 파악하려고 할 때 사용되며, 이 요인들은 관찰된 변수들 간의 상관 관계를 바탕으로 구성됩니다.

##### ❗️핵심 포인트
- 목적: 변수들 간의 관계를 바탕으로 기본적인 구조나 패턴을 찾아냄.
- 요인: 직접적으로 관측할 수 없는 잠재적인 개념(예: 신뢰성, 만족도 등)을 설명하는 변수들.
- 탐색적: 사전 가정 없이 데이터를 기반으로 요인을 추출.

### 🔥 탐색적 요인분석의 필요성
- 설문지, 심리검사 등에서 여러 문항이 하나의 개념(요인)을 측정하는지 확인
- 변수 개수를 줄여 데이터를 단순화
- 측정 도구의 구조를 이해하고 신뢰도를 높이기 위해 사용

### 🧩 탐색적 요인분석의 핵심 개념
- 요인(Factor)
여러 변수들이 공유하는 공통된 개념을 의미합니다. 예를 들어, 설문에서 "만족도", "재구매 의향", "추천 의향"이 높은 상관관계를 보인다면, 이들은 하나의 "고객 만족" 요인으로 묶일 수 있습니다.

- 공통성(Communality)
각 변수가 요인들에 의해 설명되는 정도를 나타냅니다. 0~1 사이의 값이며, 1에 가까울수록 해당 변수가 요인 구조에서 잘 설명된다는 의미입니다.

- 요인 적재값(Factor Loadings)
각 변수가 특정 요인과 얼마나 강한 연관이 있는지를 나타냅니다. 0.4 이상이면 해당 요인과 의미 있는 관계를 가진다고 판단합니다.

- 회전(Rotation)
요인 해석을 용이하게 하기 위해 축을 조정하는 과정입니다. 대표적으로 직교 회전(Varimax)과 사각 회전(Oblimin)이 있습니다.

### 🔬 탐색적 요인분석 방법
#### (1) 데이터의 적합성 평가: 변수 간 상관관계 확인
요인분석을 하려면 문항들끼리 일정 수준 이상의 상관관계가 있어야 합니다.  
→ 상관계수가 0.3 이상이면 요인분석이 가능할 가능성이 높음

#### (2) 요인 개수 결정
몇 개의 요인으로 나눌지 판단 기준은 다음과 같습니다.
- 스크리 플롯(Scree Plot): 급격히 감소하는 지점에서 요인 개수를 결정합니다.
-  설명된 분산(Explained Variance): 전체 데이터 분산의 60% 이상을 설명하는 것이 일반적으로 적절합니다. 만약 설명력이 너무 낮다면 요인 개수를 늘리거나 다른 변수 구성을 고려해야 합니다.

#### (3) 요인 추출 방법 선택
**요인추출 방법(Factor Extraction Methods)**은 변수들 간의 공통된 패턴을 기반으로 잠재 요인(Latent Factors)을 추출하는 다양한 기법이 있습니다.  

| 요인 추출 방법 | 설명 |  장점 | 단점 | 사용 사례 |
|--------------|------|------|------|--------|
| **주축 요인법 (PAF, Principal Axis Factoring)** | 공통분산(Communality)만을 이용해 요인 추출 |  요인의 해석력이 높음 | 전체 분산을 반영하지 않음 | 심리학, 사회과학 연구에서 숨겨진 요인 탐색 |
| **주성분 분석 (PCA, Principal Component Analysis)** | 전체 분산(Total Variance)을 고려하여 차원 축소 목적 |  차원 축소에 효과적 | 요인이 아닌 단순한 성분 추출 | 데이터 압축, 기계 학습 전처리 |
| **최대 우도법 (ML, Maximum Likelihood)** | 주어진 데이터가 정규분포를 따른다고 가정하고, 우도를 최대화하는 요인 추출 |  신뢰 구간 제공, 가설 검정 가능 | 정규성 가정 필요 | 구조 방정식 모델(SEM), 확인적 요인분석(CFA) |
| **가중치 없는 최소제곱 (WLS, Weighted Least Squares)** | 변수의 신뢰도를 고려하지 않고 최소제곱을 이용해 요인을 추출 | 정규성 가정 없이 분석 가능 | 작은 표본에서는 부정확할 수 있음 | 설문조사 데이터 분석 |
| **일반 최소제곱 (OLS, Ordinary Least Squares)** | 오차 제곱합을 최소화하는 방식으로 요인 추출 |  계산이 단순 | 데이터 오차가 클 경우 신뢰도 낮음 | 기초적인 탐색적 요인 분석 |
| **알파 요인 (Alpha Factoring)** | 크론바흐 알파(Cronbach’s α)를 기준으로 요인을 추출 |  신뢰도 분석과 연결 가능 | 해석이 어려울 수 있음 | 신뢰도 검증, 문항 구성 평가 |
| **최소잔차 (MinRes, Minimum Residuals)** | 잔차(Residuals)를 최소화하여 요인을 추출 |  계산이 간단하고 해석 용이 | 정확도가 높지 않을 수 있음 | 기초적인 요인 탐색 |
| **이미지 요인 (Image Factoring)** | 변수 간 상관행렬에서 추정된 공통 이미지를 이용해 요인 추출 | 변수 간 상관관계를 명확히 반영 | 데이터 특성에 따라 해석이 어려울 수 있음 | 변수들 간 관계 분석 |

##### 🔑 요약
- **PCA는 차원 축소**에 주로 사용되며, 요인 분석(Factor Analysis)보다는 **데이터 압축**에 가깝습니다.
- **PAF와 ML**은 **숨겨진 요인(공통된 개념)을 탐색하는 데 적합**합니다.
- **ML은 가설 검정이 가능하며 확인적 요인분석(CFA)과 연계**되지만, **정규성 가정이 필요**합니다.
- **WLS는 정규성을 가정하지 않기 때문에 비모수적인 환경에서 적절**합니다.
- **Alpha Factoring은 신뢰도 분석과 관련이 있어 설문지 개발에서 활용**됩니다.
- **MinRes는 단순한 요인 분석에 사용되지만, 정밀한 분석에는 적합하지 않을 수 있음**.
- **Image Factoring은 변수들 간 관계를 분석할 때 사용**되지만, 일반적인 요인분석에서는 잘 사용되지 않습니다.

#### (4) 요인 회전(Rotation) 방법 선택
요인들이 서로 독립적으로 해석되도록 회전 기법을 사용합니다. 요인 회전 방법은 다음과 같습니다.    

| 회전 방법 | 유형 | 특징 | 장점 | 단점 | 사용 사례 |
|----------|-------|------|------|------|----------|
| **비회전 (No Rotation)** | 없음 | 요인 구조를 그대로 유지 | 원래 구조를 유지 | 요인 해석이 어려울 수 있음 | 초기 분석 단계 |
| **Varimax** | 직교 회전 | 요인 간 상관이 없다고 가정하고, 변수들이 특정 요인에 강하게 부하되도록 조정 | 해석이 쉬움, 요인의 독립성을 유지 | 요인 간 상관이 있을 경우 부적절 | 심리학, 사회과학 연구에서 가장 널리 사용 |
| **Quartimax** | 직교 회전 | 각 변수들이 가능한 한 적은 요인에 부하되도록 조정 | 첫 번째 요인에 집중되어 요인 수를 줄이는 효과 | 요인들이 명확히 분리되지 않을 수 있음 | 인지능력 연구 등에서 사용 |
| **Equamax** | 직교 회전 | Varimax와 Quartimax의 절충 | 요인의 독립성과 단순 구조를 동시에 고려 | 특정 요인에 편중될 가능성 | 다양한 분야에서 사용 가능 |
| **Promax** | 사교 회전 | 요인 간 상관을 허용하는 비선형 회전 | 대규모 데이터에서도 빠르게 계산 가능 | 요인 간 상관이 높아질 수 있음 | 심리학, 사회과학 연구에서 많이 사용 |
| **Oblimin** | 사교 회전 | 요인 간 상관을 허용하는 선형 회전 | 요인 간 관계를 반영 가능 | 해석이 어려울 수 있음 | 인과관계가 있는 데이터 분석 |

##### 📏 선택기준
요인 간 상관이 없다고 가정하면 → 직교 회전 (Varimax, Quartimax, Equamax)
- Varimax: 가장 많이 사용되며 요인을 명확히 구분
- Quartimax: 첫 번째 요인에 집중되도록 조정 (주로 인지능력 연구)
- Equamax: Varimax와 Quartimax의 절충

요인 간 상관이 있을 가능성이 크다면 → 사교 회전 (Promax, Oblimin)
- Promax: 계산이 빠르고 대규모 데이터에 적합
- Oblimin: 요인 간 관계를 자연스럽게 반영

 <div style="border: 1px solid #FFD700; padding: 15px; border-radius: 5px; background-color: #FFFACD; color: #333;">
 💡 일반적으로 Varimax가 기본 선택이며, 요인 간 상관이 필요할 경우 Promax나 Oblimin을 고려합니다.

#### 4. 신뢰성 및 타당성 평가
##### (1) 요인 점수
요인 점수(Factor Score)는 각 개체가 특정 요인에서 가지는 상대적인 값을 수치로 나타낸 것입니다.   
종류는 다음과 같이 3가지로 구분됩니다. 
- 회계계수(Correlation of (regression) scores with factors): 요인 점수와 해당 요인 간의 상관계수를 나타냅니다.
- 결정계수(Multiple R square of scores with factors): 요인 점수가 해당 요인을 얼마나 잘 설명하는지를 나타냅니다. 값이 1에 가까울수록 설명력이 높습니다.
- 가능한 요인 점수 간의 최소 상관(Minimum correlation of possible factor scores): 요인 점수 간의 최소 상관을 나타내며, 이는 요인 점수의 신뢰성을 평가하는 데 활용될 수 있습니다. 

##### 📝 해석
- MR1: 매우 높은 상관 계수와 R² 값(각각 0.9048, 0.8186)을 가지며, 회귀 점수와의 관계가 매우 강합니다. 최소 상관 값은 0.6372로 중간 수준입니다.
- MR2: 회귀 점수와 매우 강한 상관 관계(0.9192)와 높은 R² 값(0.8448)을 보이며, 최소 상관 값은 0.6897로 상대적으로 높은 수준입니다.
- MR3: 상관 계수와 R² 값이 0.8928과 0.7971로 상당히 높으며, 최소 상관 값은 0.5943으로 상대적으로 낮습니다.

##### (2) Scree plot과 적재값(factor loading) 히트맵 
Scree plot은 요인의 수를 결정하는 데 도움을 주며, 고유값을 시각적으로 확인할 수 있습니다.  
Factor Loading Heatmap은 각 요인과 변수들 간의 관계를 시각적으로 파악할 수 있게 해주며, 각 요인의 적재값을 색상으로 표현하여 관계의 강도를 쉽게 비교할 수 있게 합니다.
