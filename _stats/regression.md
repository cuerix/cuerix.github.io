# 회귀분석(Regression Analysis)이란?

회귀분석(Regression Analysis)은 변수들 간의 관계를 분석하고, 한 변수(종속변수, Y)가 다른 변수(독립변수, X)의 변화에 따라 어떻게 변하는지를 예측하는 통계 기법입니다. 회귀분석은 경제학, 경영학, 심리학, 공학 등 다양한 분야에서 데이터 분석 및 의사결정에 널리 사용됩니다.

이 글에서는 회귀분석의 개념, 유형, 가정, 해석 방법, 그리고 실제 분석 예제를 살펴보겠습니다.

---

## 1. 회귀분석의 개념

회귀분석은 주어진 데이터에서 변수 간의 수학적 관계를 찾아내고 이를 이용해 미래 값을 예측하거나 변수들의 영향력을 측정하는 방법입니다. 예를 들어, "광고비가 매출에 미치는 영향은 어떠한가?" 또는 "공부 시간이 시험 성적에 어떤 영향을 주는가?" 같은 질문에 답하기 위해 사용됩니다.

회귀분석에서 가장 기본적인 형태는 선형 회귀(linear regression)로, 변수 간의 관계를 직선 형태로 모델링합니다.

기본적인 선형 회귀 방정식은 다음과 같습니다.

\(Y = \beta_0 + \beta_1X + \epsilon\)

- \(Y\): 종속변수 (예측하고 싶은 변수)
- \(X\): 독립변수 (종속변수에 영향을 주는 변수)
- \(\beta_0\): 절편 (Intercept)
- \(\beta_1\): 기울기 (Coefficient, 계수)
- \(\epsilon\): 오차항 (Error Term, 예측이 완벽하지 않기 때문에 존재하는 오차)

즉, \(X\)가 한 단위 증가할 때 \(Y\)가 \(\beta_1\)만큼 변화한다고 해석할 수 있습니다.

---

## 2. 회귀분석의 주요 유형

회귀분석에는 다양한 유형이 있으며, 분석 목적과 데이터 특성에 따라 적절한 방법을 선택해야 합니다.

### 2.1 단순선형회귀 (Simple Linear Regression)

독립변수가 하나일 때 사용하는 회귀분석 기법입니다. 예를 들어, 광고비(X)와 매출(Y)의 관계를 분석하는 경우가 이에 해당합니다.

\(Y = \beta_0 + \beta_1X + \epsilon\)

### 2.2 다중선형회귀 (Multiple Linear Regression)

독립변수가 여러 개일 때 사용하는 방법으로, 현실에서는 단순 회귀보다 더 많이 사용됩니다. 예를 들어, 매출(Y)에 영향을 미치는 요소가 광고비(X1), 제품 가격(X2), 브랜드 인지도(X3) 등 여러 개라면 다음과 같은 다중 회귀 모델을 사용할 수 있습니다.

\(Y = \beta_0 + \beta_1X_1 + \beta_2X_2 + \beta_3X_3 + \epsilon\)

### 2.3 로지스틱 회귀 (Logistic Regression)

종속변수가 범주형(예: 구매 여부 - Yes/No)일 때 사용하는 회귀분석 방법입니다. 일반 선형 회귀와 달리, 로지스틱 함수(시그모이드 함수)를 사용하여 예측 확률을 계산합니다.

### 2.4 다항 회귀 (Polynomial Regression)

독립변수와 종속변수의 관계가 직선이 아닌 곡선 형태일 때 사용하는 방법입니다. 예를 들어, 시험 성적과 공부 시간의 관계가 단순 직선이 아니라 일정 시간이 지나면 효과가 감소하는 패턴이라면 다항 회귀를 적용할 수 있습니다.

\(Y = \beta_0 + \beta_1X + \beta_2X^2 + \epsilon\)

### 2.5 시계열 회귀 (Time Series Regression)

시간(t)에 따라 변하는 데이터를 분석할 때 사용하는 방법으로, 주식 가격, 매출 예측 등에서 활용됩니다.

---

## 3. 회귀분석의 가정

회귀분석을 수행하기 위해서는 몇 가지 기본 가정을 충족해야 합니다. 이를 만족하지 않으면 회귀 분석의 결과가 왜곡될 수 있습니다.

1. **선형성 (Linearity)**: 독립변수와 종속변수 간의 관계가 선형이어야 합니다.
2. **독립성 (Independence)**: 오차항(ε)들은 서로 독립적이어야 합니다.
3. **등분산성 (Homoscedasticity)**: 오차의 분산이 일정해야 합니다.
4. **정규성 (Normality)**: 오차항이 정규분포를 따라야 합니다.
5. **다중공선성 없음 (No Multicollinearity)**: 독립변수들 간에 높은 상관관계가 없어야 합니다.

---

## 4. 회귀분석 결과 해석

회귀분석을 수행한 후에는 결과를 해석해야 합니다. 주요 해석 요소는 다음과 같습니다.

1. **회귀계수(β, Coefficients)**: 독립변수가 종속변수에 미치는 영향의 크기를 나타냅니다.
2. **R² (결정계수, Coefficient of Determination)**: 모델의 설명력을 나타내며, 0에서 1 사이의 값을 가집니다. 1에 가까울수록 좋은 모델입니다.
3. **p-값 (p-value)**: 독립변수가 종속변수에 통계적으로 유의미한 영향을 주는지 확인합니다. 일반적으로 p-value가 0.05 이하이면 유의미하다고 판단합니다.
4. **잔차 분석 (Residual Analysis)**: 가정이 제대로 충족되었는지 확인하는 과정입니다.

## 5. 실전 예제 (R 또는 Python 코드 활용)

### Python을 이용한 단순 회귀 분석 예제

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm

# 샘플 데이터 생성
data = pd.DataFrame({
    '광고비': [10, 20, 30, 40, 50],
    '매출': [5, 9, 15, 20, 25]
})

X = data['광고비']
Y = data['매출']
X = sm.add_constant(X)  # 절편 추가

# 회귀 분석 실행
model = sm.OLS(Y, X).fit()
print(model.summary())
```

이 코드를 실행하면 회귀계수, p-value, R² 등의 분석 결과를 얻을 수 있습니다.
