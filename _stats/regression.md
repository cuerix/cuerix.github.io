## 개요

회귀분석은 통계학에서 매우 중요한 분석 기법으로, <u>변수들 간의 관계를 모델링하고 예측</u>하는 데 사용됩니다. 주로 종속 변수와 독립 변수 간의 관계를 파악하고 예측 모델을 만들 때 사용되며, 다양한 분야에서 활용됩니다. 이 글에서는 회귀분석의 기본 개념부터 시작해, 그 유형과 활용법에 대해 자세히 알아보겠습니다.

### 1. 회귀분석의 개념

회귀분석(Regression Analysis)은 두 개 이상의 변수 간의 관계를 모델링하는 기법입니다. 여기서 '회귀'는 독립 변수들이 종속 변수에 미치는 영향을 파악하는 과정입니다. <u>종속 변수는 예측하고자 하는 대상</u>이며, <u>독립 변수는 그 예측에 영향을 미칠 수 있는 변수</u>들입니다.

- 목적: 변수 간의 관계를 모델링하여 예측하거나 설명하는 것입니다. 
- 독립 변수: 예측에 영향을 미치는 변수들입니다. 
- 종속 변수: 우리가 예측하고 싶은 값으로, 독립 변수들에 의해 영향을 받습니다.


### 2. 회귀분석의 기본 가정
회귀분석을 수행할 때 모델이 성립하귀 위해 필요한 가정이 있습니다. 이러한 가정들이 충족되지 않으면 분석 결과가 왜곡될 수 있습니다. 
주요 가정은 다음과 같습니다.

#### 선형성 (Linearity)
- 독립변수와 종속변수 사이에는 <u>선형적인 관계</u>가 있어야 합니다. 즉, 변수들이 직선적인 관계로 연결되어야 한다는 의미입니다.
- 검토 방법: <span style="background-color:#fff5b1"> 산점도(scatter plot)</span>나 <span style="background-color:#fff5b1"> 잔차 플롯(residual plot)</span>을 통해 선형 관계를 확인할 수 있습니다.
#### 독립성 (Independence)
- 각 관측치는 <u>서로 독립적</u>이어야 합니다. 즉, 한 데이터 포인트가 다른 데이터 포인트에 영향을 미치지 않아야 합니다.
- 검토 방법:<span style="background-color:#fff5b1">더미 변수</span>나 <span style="background-color:#fff5b1">Durbin-Watson 테스트</span>를 통해 독립성을 확인할 수 있습니다.
#### 등분산성 (Homoscedasticity)
- 잔차(실제 값과 예측 값의 차이)의 분산이 일정해야 합니다. 즉, 예측 값이 커지거나 작아져도 잔차의 분산이 일정해야 한다는 의미입니다.
- 검토 방법: 잔차 플롯에서 <span style="background-color:#fff5b1">잔차의 분산</span>이 일정한지 확인합니다. 만약 잔차가 특정 패턴을 보이면 이 가정이 위반된 것입니다.
#### 정규성 (Normality)
- 잔차들이 정규분포를 따라야 합니다. 이는 회귀 분석의 유의성 검정을 정확하게 수행하기 위한 조건입니다.
- 검토 방법: <span style="background-color:#fff5b1">히스토그램</span>이나 <span style="background-color:#fff5b1">Q-Q 플롯</span>을 사용하여 잔차가 정규분포를 따르는지 확인할 수 있습니다.
#### 다중공선성 (Multicollinearity)
- 다중회귀분석에서 여러 독립변수 간에 강한 상관관계가 없어야 합니다. 두 변수 이상의 독립변수가 매우 높은 상관관계를 가지면, 회귀계수가 불안정해질 수 있습니다.
- 검토 방법: span style="background-color:#fff5b1">분산팽창계수(VIF, Variance Inflation Factor)</span>를 계산하여 다중공선성을 확인할 수 있습니다. VIF 값이 10을 초과하면 다중공선성 문제가 있을 수 있습니다.



### 3. 회귀분석의 유형

#### 선형 회귀분석 (Linear Regression)
가장 기본적인 회귀분석으로, 독립 변수와 종속 변수 간의 선형 관계를 모델링합니다. 단순 선형 회귀는 독립 변수가 하나일 때, 다중 선형 회귀에서는 독립 변수가 여러 개일 때 사용됩니다.

#### 로지스틱 회귀분석 (Logistic Regression)
로지스틱 회귀는 종속 변수가 <span style="background-color:#fff5b1">범주형</span>일 때 사용됩니다. 예를 들어, 고객이 제품을 구매할지 말지를 예측하는 모델에서 구매 여부(구매/비구매)를 예측할 수 있습니다. 이때 종속 변수는 <span style="background-color:#fff5b1">`이진`(0 또는 1) 값</span>을 갖습니다. 로지스틱 회귀는 '로짓 함수'를 사용해 확률을 예측합니다.

#### 매개효과 (Mediation Effect)
매개효과는 독립 변수가 종속 변수에 미치는 영향이 다른 중간 변수인 `매개 변수` 통해 전달되는 과정을 설명하는 개념입니다. 즉, 독립 변수가 종속 변수에 직접적인 영향을 미치지 않고, 매개 변수를 통해 간접적으로 영향을 미치는 경우입니다.

> ##### 📋 Note
>매개효과는 <span style="background-color:#fff5b1">Baron & Kenny (1986)**의 4단계 검증법</span>이나 <span style="background-color:#fff5b1">부트스트랩(bootstrapping) 방법</span>을 통해 확인할 수 있습니다. 이를 통해 매개변수가 독립변수와 종속변수 간의 관계를 얼마나 강하게 중재하는지 파악할 수 있습니다.

#### 조절효과 (Moderation Effect)
조절효과는 독립 변수와 종속 변수 간의 관계가 특정한 변수(조절 변수)에 의해 달라지는 경우를 설명하는 개념입니다. 즉, 조절 변수는 독립 변수와 종속 변수 간의 관계의 강도나 방향을 변화시키는 역할을 합니다.


## 4. 회귀분석 결과 해석

회귀분석을 수행한 후에는 결과를 해석해야 합니다. 주요 해석 요소는 다음과 같습니다.

1. **회귀계수(β, Coefficients)**: 독립변수가 종속변수에 미치는 영향의 크기를 나타냅니다.
2. **R² (결정계수, Coefficient of Determination)**: 모델의 설명력을 나타내며, 0에서 1 사이의 값을 가집니다. 1에 가까울수록 좋은 모델입니다.
3. **p-값 (p-value)**: 독립변수가 종속변수에 통계적으로 유의미한 영향을 주는지 확인합니다. 일반적으로 p-value가 0.05 이하이면 유의미하다고 판단합니다.
4. **잔차 분석 (Residual Analysis)**: 가정이 제대로 충족되었는지 확인하는 과정입니다.

## 5. 실전 예제 (R 또는 Python 코드 활용)

### Python을 이용한 단순 회귀 분석 예제

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm

# 샘플 데이터 생성
data = pd.DataFrame({
    '광고비': [10, 20, 30, 40, 50],
    '매출': [5, 9, 15, 20, 25]
})

X = data['광고비']
Y = data['매출']
X = sm.add_constant(X)  # 절편 추가

# 회귀 분석 실행
model = sm.OLS(Y, X).fit()
print(model.summary())
```

이 코드를 실행하면 회귀계수, p-value, R² 등의 분석 결과를 얻을 수 있습니다.

## 6. 결론

회귀분석은 데이터 간의 관계를 분석하고 예측하는 데 강력한 도구입니다. 하지만 기본 가정을 만족하는지 확인하고, 적절한 유형을 선택하는 것이 중요합니다. 또한, 회귀 계수 해석과 모델 평가를 통해 신뢰성 있는 결론을 도출해야 합니다.
