## 상세 개요
군집분석(Clustering Analysis)은 비지도 학습(Unsupervised Learning) 방법 중 하나로, 유사한 데이터들을 자동으로 그룹화하는 기법입니다. 이는 데이터 내 숨겨진 패턴을 탐색하고, 특정한 기준 없이도 자연스러운 분류를 수행할 수 있도록 합니다.

## 군집분석의 개념
군집분석의 기본 원리는 다음과 같습니다.

- 유사성이 높은 데이터들을 하나의 군집(cluster)으로 묶음

- 데이터 간의 유사성을 측정하여 그룹화
보통 거리(유클리드 거리 등)를 기준으로 유사성을 판단
각 군집이 서로 뚜렷이 구별될 수 있도록 설정

- 같은 군집 내 데이터 간 거리는 가깝게
서로 다른 군집 간 거리는 멀게
이 과정에서 데이터가 어떠한 사전 정의된 레이블 없이도 자동으로 분류되므로, 지도학습(Supervised Learning)과 다릅니다.

2. 군집분석의 주요 목적
군집분석은 여러 분야에서 활용될 수 있으며, 대표적인 목적은 다음과 같습니다.

✅ 1) 데이터 구조 탐색
군집을 형성함으로써 데이터의 전반적인 패턴을 파악할 수 있습니다. 이를 통해 연구자나 분석가가 숨겨진 특성을 발견할 수 있습니다.

✅ 2) 고객 세분화(Customer Segmentation)
마케팅에서 고객을 소비 패턴, 선호도 등에 따라 그룹화하여 타겟 마케팅을 수행할 수 있습니다.
예: 쇼핑몰에서 구매 데이터를 기반으로 VIP 고객, 일반 고객, 신규 고객으로 세분화
✅ 3) 이상 탐지(Anomaly Detection)
정상적인 데이터 패턴과 큰 차이를 보이는 데이터를 탐지하는 데 활용됩니다.
예: 금융 거래에서 신용카드 사기 탐지
✅ 4) 차원 축소(Dimensionality Reduction)
다차원 데이터를 군집화하여 복잡도를 줄이고 주요 특징을 유지하는 방식으로 사용됩니다.
예: 문서 데이터에서 주요 주제별로 그룹을 형성
✅ 5) 이미지 및 텍스트 분류
비슷한 색상, 형태를 가진 이미지를 자동으로 분류
뉴스 기사나 문서를 특정 주제별로 자동 군집화
3. 군집분석의 주요 기법 및 알고리즘
군집분석을 수행하는 방법은 여러 가지가 있으며, 대표적인 알고리즘을 소개합니다.

1) K-평균 군집(K-Means Clustering)
✔ 개념

데이터를 K개의 군집으로 나누는 알고리즘
각 군집의 중심점(centroid)을 설정하고, 데이터 포인트를 가장 가까운 중심점에 할당
반복적으로 중심점을 조정하여 군집을 최적화
✔ 장점

계산 속도가 빠름
대규모 데이터에서 효율적
✔ 단점

K값(군집 개수)을 미리 정해야 함
이상치(Outlier)에 민감
✔ 활용 예시

고객 세분화
이미지 색상 분할
2) 계층적 군집분석(Hierarchical Clustering)
✔ 개념

데이터 간 계층적인 구조를 형성하며 군집을 만들음
덴드로그램(Dendrogram)이라는 트리 형태의 구조로 표현
병합적(agglomerative) 방식과 분할적(divisive) 방식이 있음
✔ 장점

K값(군집 개수)을 미리 설정할 필요 없음
덴드로그램을 통해 군집 관계를 쉽게 시각화 가능
✔ 단점

데이터 수가 많아지면 계산량이 급격히 증가
✔ 활용 예시

유전자 분석(비슷한 유전자를 군집화)
문서 분류
3) DBSCAN(Density-Based Spatial Clustering of Applications with Noise)
✔ 개념

밀집도(Density)를 기준으로 군집을 형성하는 알고리즘
밀도가 높은 지역을 군집으로 묶고, 밀도가 낮은 지역은 이상치로 간주
✔ 장점

이상치에 강한 군집화 가능
다양한 형태의 군집을 탐색 가능
✔ 단점

데이터의 밀도 기준(ε, MinPts)을 설정하는 것이 어렵다
✔ 활용 예시

지리적 데이터 분석(GPS 데이터 군집화)
이상 거래 탐지
4) GMM(Gaussian Mixture Model, 가우시안 혼합 모델)
✔ 개념

데이터를 여러 개의 정규분포(Gaussian Distribution)로 모델링하여 군집을 찾음
각 데이터 포인트가 여러 군집에 속할 확률을 계산
✔ 장점

데이터가 복잡한 분포를 가질 때 유용
유연한 군집화 가능
✔ 단점

계산량이 많아 대규모 데이터에서는 속도가 느림
✔ 활용 예시

음성 신호 분석
패턴 인식
4. 군집분석 평가 방법
군집분석 결과가 좋은지 평가하기 위해 다음과 같은 지표를 사용합니다.

1) 실루엣 계수(Silhouette Score)
각 데이터 포인트가 속한 군집 내에서 얼마나 밀집되어 있는지 평가
값이 1에 가까울수록 좋은 군집화
2) Dunn Index
군집 간 거리와 군집 내 거리의 비율을 측정하여 군집의 품질 평가
3) SSE(Sum of Squared Errors, 오류 제곱합)
군집의 중심점과 각 데이터 포인트 간의 거리 합을 계산
값이 작을수록 군집화가 잘 이루어짐
5. 군집분석 실전 활용 예시
📌 1) 마케팅 & 고객 세분화
소비자 구매 패턴을 기반으로 VIP, 일반, 신규 고객으로 그룹화
개별 고객군에 맞는 마케팅 전략 적용
📌 2) 이상 탐지(Anomaly Detection)
금융권에서 신용카드 사기 탐지
네트워크 트래픽 분석을 통한 비정상적인 패턴 탐지
📌 3) 의료 데이터 분석
환자의 유전자 데이터를 분석하여 질병 유형 분류
건강검진 데이터를 활용해 질병 위험군을 군집화
📌 4) 텍스트 마이닝 & 문서 군집화
뉴스 기사를 주제별로 자동 군집화
고객 리뷰 데이터를 분석하여 주요 키워드 기반으로 그룹화
6. 결론
군집분석은 비지도 학습의 대표적인 기법으로, 데이터의 숨겨진 패턴을 발견하고 활용하는 데 유용합니다. 다양한 알고리즘이 있으며, 데이터의 특성에 따라 적절한 방법을 선택하는 것이 중요합니다.

🔥 군집분석을 잘 활용하면 마케팅, 금융, 의료, 텍스트 분석 등 다양한 분야에서 데이터 기반 의사결정을 효과적으로 내릴 수 있습니다!


ward.D / ward.D2: 군집 내 분산 증가 최소화 (Ward 방식)
single: 가장 가까운 점(최소 거리) 기준
complete: 가장 먼 점(최대 거리) 기준
average: 모든 점 간 평균 거리 기준
centroid: 군집 중심 간 거리 기준
mcquitty: 가중 평균 거리 기준
median: 군집 중앙점 간 거리 기준

euclidean: 두 점 간 직선 거리 (L2 거리)
maximum: 두 차원 간 최대 거리 (체비쇼프 거리)
manhattan: 차원별 절대 거리 합 (L1 거리)
canberra: 차원별 비율 기반 거리 (가중 거리)
binary: 이진 데이터의 불일치율 기반 거리
minkowski: 일반화된 거리 측정 (p-노름 거리)
