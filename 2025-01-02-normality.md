## 개요

정규성 검정(Normality Test)은 <span style="background-color:#fff5b1">데이터가 정규분포를 따르는지 여부를 확인</span>하는 통계적 검정입니다. 정규분포는 데이터가 평균을 중심으로 좌우가 대칭적이고, 중앙에 가장 많은 값이 몰리고 양쪽으로 갈수록 값이 적게 분포하는 형태의 분포입니다. 많은 통계 분석 기법, 특히 평균과 표준편차를 사용하는 기법들은 데이터가 정규분포를 따른다는 가정하에 성립합니다. 따라서, 정규성 검정은 데이터가 이러한 가정을 만족하는지 확인하는 과정입니다.

### ❓ 정규분포란?
정규분포는 연속 확률 분포의 일종으로, 가장 널리 알려진 분포입니다. 평균값을 중심으로 좌우 대칭적으로 분포하고, 특정 수학적 성질을 가집니다. 정규분포를 따르는 데이터는 68-95-99.7 규칙을 따릅니다:

- 데이터의 약 68%가 평균 ± 1 표준편차 내에 존재
- 약 95%가 평균 ± 2 표준편차 내에 존재
- 약 99.7%가 평균 ± 3 표준편차 내에 존재


### 🔥 정규성 검정의 필요성
많은 통계 분석에서 데이터가 정규분포를 따른다고 가정합니다. 예를 들어, <span style="background-color:#fff5b1">t-검정, ANOVA, 회귀분석 등</span>에서는 데이터가 정규분포를 따른다는 가정이 중요한 역할을 합니다. 이 가정을 만족하지 않으면, 분석 결과가 왜곡될 수 있기 때문에 정규성 검정을 통해 데이터의 분포를 확인하는 것이 중요합니다.    


### 🔬 검정 방법
정규성 검정을 수행하는 방법은 여러 가지가 있으며, 대표적인 방법은 다음과 같습니다:

#### 📈 시각화
- **히스토그램**: 데이터의 분포를 직관적으로 확인할 수 있습니다. 정규분포에 가까운 형태를 보이면 정규분포를 따를 가능성이 있습니다.
- **Q-Q 플롯 (Quantile-Quantile Plot)**: 이 그래프는 데이터의 분위수를 이론적인 정규분포의 분위수와 비교합니다. 만약 점들이 직선 형태에 가까우면 데이터가 정규분포를 따른다고 볼 수 있습니다.

#### 🔢 통계적 검정
정규성 검정을 위해 여러 통계적 방법이 사용됩니다. 주요 방법은 다음과 같습니다:

#### 샤피로-윌크 검정(Shapiro-Wilk Test)
  특정 데이터셋이 정규 분포를 따르는지 검정하는 방법으로 특히 샘플 크기가 작을 때(50 이하) 강력한 성능을 보입니다.

##### ❗️핵심 포인트
- 귀무가설 (H0): 데이터가 정규 분포를 따른다.
- 대립가설 (H1): 데이터가 정규 분포를 따르지 않는다.
- 해석:
  - p-값 > 0.05 → 정규성을 만족한다고 볼 수 있음 ✅
  - p-값 ≤ 0.05 → 정규성을 만족하지 않음 ❌
- 사용 예시: 정규성 검정(ANOVA, t-검정 등에서 필수)

#### 단일 표본 콜모고로프-스미르노프 검정(One-Sample Kolmogorov-Smirnov Test)
  이 검정은 데이터가 특정 분포(보통 정규분포)를 따르는지 확인하기 위해 누적분포함수(CDF)를 비교하는 방식입니다.

##### ❗️핵심 포인트
- 귀무가설 (H0): 데이터가 특정 분포를 따른다.
- 대립가설 (H1): 데이터가 특정 분포를 따르지 않는다.
- 해석:
  - p-값 > 0.05 → 특정 분포를 따른다고 볼 수 있음 ✅
  - p-값 ≤ 0.05 → 특정 분포를 따르지 않음 ❌
- 한계점: 정규성 검정에 있어 Shapiro-Wilk 검정보다 덜 강력함
- 사용 예시: 데이터가 특정 이론적 분포(정규, 균등, 지수 등)를 따르는지 검정할 때 사용

#### 이표본 콜모고로프-스미르노프 검정(Two-Sample Kolmogorov-Smirnov Test)
  두 개의 데이터 집합이 동일한 분포를 가지는지 확인하는 데 사용됩니다. 이를 통해 두 집단 간의 분포 차이를 검정할 수 있습니다.
  
##### ❗️핵심 포인트
- 귀무가설 (H0): 두 개의 데이터는 동일한 분포를 따른다.
- 대립가설 (H1): 두 개의 데이터는 서로 다른 분포를 따른다.
- 해석:
  - p-값 > 0.05 → 두 분포가 동일하다고 볼 수 있음 ✅
  - p-값 ≤ 0.05 → 두 분포가 다를 가능성이 높음 ❌
- 사용 예시: 실험군 vs 대조군 데이터 비교, 시간에 따른 분포 변화 분석

### ⚠️ 정규성을 따르지 않은 경우 
#### 데이터 변환
변수관리에서 로그 변환, 제곱근 변환, 박스-콕스(Box-Cox) 등 데이터를 변환하여 다시 분석을 실시하실 수 있습니다.
#### 비모수 검정 사용
Mann-Whitney U, Kruskal-Wallis, Wilcoxon 등 비모수 검정을 활용하실 수 있습니다.

보다 자세한 내용은 데이터 변환, 비모수 검정에서 확인하세요.
