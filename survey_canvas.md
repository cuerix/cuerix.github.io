### Survey Canvas  
설문지를 디자인하기 위한 도구 모음입니다. 사용자는 다양한 설문도구를 필요에 따라 꺼내 문항을 디자인할 수 있습니다. Canvas는 컴퓨터 모니터와 휴대전화 크기로 변경해 설정할 수 있으며, 더 상세한 설정은 설정 가능을 활용하세요.

### 설문 도구 설정 기능

#### 기본 설문 도구  
![Image](https://github.com/user-attachments/assets/ccd4ccae-4e7b-48d6-aaba-001c7b5bf7cc)  

  

| 유형       | 형식          | 목적 / 용도 |
|------------|--------------|---------------------------------------------|
| **1개 선택**  | 라디오 버튼  | 응답자가 하나의 답변만 선택하도록 함. 선호도, 만족도 조사 등에 사용됨. 명확한 선호도를 측정하는 데 유용하며, 배타적인 선택이 필요한 경우 적합함. |
| **다중 선택**  | 체크박스      | 여러 개의 답변을 선택할 수 있도록 함. 사용 경험, 취향 조사 등에 활용됨. 응답자가 복수의 항목을 선택할 수 있어야 할 때 적절함. |
| **수치**       | 숫자 입력란  | 정량적인 데이터를 수집할 때 사용됨. 연령, 점수, 금액 등의 입력이 필요할 때 활용됨. 데이터 분석 및 통계 처리가 필요한 경우 유용함. |
| **텍스트**     | 텍스트 입력란 | 응답자의 자유로운 의견이나 설명을 받을 때 사용됨. 개방형 응답을 수집할 때 활용되며, 주관적 의견이나 상세한 피드백을 요구할 때 적합함. 다만, 분석이 어렵기 때문에 필요할 때만 사용을 권장함. |
| **드롭다운**   | 드롭다운 목록 | 여러 옵션 중 하나를 선택하도록 함. 선택지가 많을 때 설문을 간결하게 유지하는 데 사용됨. 선택 항목이 많을 경우 화면을 깔끔하게 정리하는 데 유용하며, 응답자가 하나의 답변만 선택하도록 제한할 수 있음. |


  

## 문항 유형 선택 가이드  
- **1개 선택 vs. 다중 선택**: 하나의 답을 강제해야 하는 경우(예: 성별, 가장 선호하는 브랜드 등)에는 **1개 선택**을, 응답자가 여러 개를 선택할 수 있어야 하는 경우(예: 사용하는 SNS 목록)에는 **다중 선택**을 사용합니다.  
- **수치형 문항**: 연령, 소득, 사용 빈도 등 **정량적인 응답**이 필요한 경우 사용합니다.  
- **텍스트 응답**: 응답자의 **자유로운 의견**을 수집할 때 사용합니다. 하지만 분석이 어렵기 때문에 필요할 때만 사용을 권장합니다.  
- **드롭다운 목록**: 선택지가 많을 경우 **설문을 간결하게 유지**하기 위해 사용합니다.

# 설문 문항 도구 설정

설문을 설계할 때 문항 유형을 적절하게 선택하는 것은 응답자의 편의성과 데이터의 정확성을 높이는 데 중요합니다.  
아래 표는 설문에서 자주 사용되는 **1개 선택, 다중 선택, 수치, 텍스트, 드롭다운** 유형의 형식, 목적/용도, 예제를 정리한 것입니다.
