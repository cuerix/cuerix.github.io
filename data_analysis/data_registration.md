**데이터 등록**은 설문 조사에서 수집된 데이터를 엑셀 파일 형식으로 시스템에 업로드하는 과정입니다. 사용자는 엑셀 파일을 통해 데이터를 손쉽게 등록하고, 이를 기반으로 분석을 진행할 수 있습니다. 엑셀 파일을 업로드하면 시스템에서 자동으로 데이터를 읽어 들여, 해당 데이터가 분석에 적합한 형태로 처리됩니다. 이를 통해 사용자는 수집된 데이터를 빠르고 효율적으로 관리하고, 분석을 위한 기초 데이터를 준비할 수 있습니다.

#### ✔️ 데이터 등록 방법
분석 대상 데이터 제목을 입력하고, 스프레드시트 파일(.xlsx)를 업로드하여 등록합니다.<br>

<div style="border: 1px solid #FFD700; padding: 15px; border-radius: 5px; background-color: #FFFACD; color: #333;">
 💡 등록한 데이터를 분석하려면 '데이터 분석'에서 Data Analyzer로 이동하세요.
</div>
